## QUIZ

- Which of the following is the correct syntax?
  - 다음 중 올바른 구문은 어떤 것인가?

> [ ] echo "(<p>Hello.</p>);
>
> [ ] echo (<p>' Hello.'</p>);
>
> [ ] `echo "<p>Hello.</p>";`

<br>

- Fill in the blank to make the text a comment:
  - 텍스트를 주석으로 만들어라.

```php
// This is a comment
echo 'This is a test';
```

- Drag and drop from the options below to form the correct PHP code:
  - 올바른 PHP 코드를 만들어라.
