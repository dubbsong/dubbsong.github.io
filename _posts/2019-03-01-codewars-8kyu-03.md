#### Problem

- Given a number, find its opposite.
  - 주어진 숫자의 반대수(opposite of a number)를 반환한다.

<br>

#### Solution 01

```js
function opposite(n) {
  return -n;
}

opposite(0);	// 0
opposite(2);	// -2
opposite(-4);	// 4
```

<br>

#### Solution 02

```js
function opposite(n) {
  return n * -1;
}

opposite(0);	// 0
opposite(2);	// -2
opposite(-4);	// 4
```

<br>

#### Solution 03

```js
function opposite(n) {
  return n / -1;
}

opposite(0);	// 0
opposite(2);	// -2
opposite(-4);	// 4
```

<br>

#### Solution 04

```js
function opposite(n) {
  return n > 0 ? -n : Math.abs(n);
}

opposite(0);	// 0
opposite(2);	// -2
opposite(-4);	// 4
```

> `Math.abs()` 메소드
>
> 주어진 숫자의 절대값(absolute value)을 반환한다.
