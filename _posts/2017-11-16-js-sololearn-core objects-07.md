## QUIZ

- Given the array below, please complete the expression to be alerted with "apple".
  - 아래 주어진 배열에서, "apple"로 alert 할 표현식을 완성해라.

```js
var fruits = new Array("pear", "orange", "apple", "grapefruit");

alert(fruits[2]);
```
