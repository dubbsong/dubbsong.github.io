#### 반복문(for loop) + 조건문(if) + 배열(array) 알고리즘 문제

<br>

## problem

Write a function '`max`'

that accepts an array of numbers

and returns the largest number in the array.

<br>

## solution 01

```javascript
function max(array) {
   let maxNum = array[0];
   
   for (let i = 0; i < array.length; i++) {
      if (maxNum < array[i]) {
         maxNum = array[i];
      }
   }
   return maxNum;
}


max([3, 2, 7, 5]);	// 7
```

<br>

## solution 02

```javascript
function max(array) {
   let maxNum = array[0];
   
   for (let i of array) {
      if (maxNum < i) {
         maxNum = i;
      }
   }
   return maxNum;
}


max([3, 2, 7, 5]);	// 7
```
