#### 반복문(for loop) + 조건문(if) + 배열(array) 알고리즘 문제

<br>

## problem

Write a function called '`min`'

that finds the smallest number in an array of numbers.

<br>

## solution 01

```javascript
function min(array) {
   let minNum = array[0];
   
   for (let i = 0; i < array.length; i++) {
      if (minNum > array[i]) {
         minNum = array[i];
      }
   }
   return minNum;
}


min([3, 2, 7, 5]);	// 2
```

<br>

## solution 02

```javascript
function min(array) {
   let minNum = array[0];
   
   for (let i of array) {
      if (minNum > i) {
         minNum = i;
      }
   }
   return minNum;
}
```
