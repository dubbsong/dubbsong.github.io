## Quiz

- What is the output of this code?

```python
>>> spam = "7"
>>> spam = spam + "0"
>>> eggs = int(spam) + 3
>>> print(float(eggs))
73.0
```

<br>

- What is the output of this code?

```python
>>> word = input("Enter a word: ")

>>> print(word + ' shop')

```
