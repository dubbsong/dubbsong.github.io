#### 반복문(for loop) 알고리즘 문제

<br>

## problem

Summation to n:

Let's implement the function sum

that takes a single parameter n,

and computes the sum of all integers

up to n starting from 0.

<br>

## solution

```javascript
function sum(n) {
   let result = 0;
   
   for (let i = 0; i <= n; i++) {
      result += i;
   }
   return result;
}


sum(3);		// 6
sum(4);		// 10
sum(100);	// 5050
```
